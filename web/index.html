常见问题

                    

                        

                            
如何添加新站点？

                            
1. 进入"站点管理"页面

                            
2. 填写域名和端口信息

                            
3. 选择是否启用SSL

                            
4. 点击"添加站点"按钮

                        

                        

                            
如何申请SSL证书？

                            
1. 进入"SSL证书"页面

                            
2. 输入需要申请证书的域名

                            
3. 点击"申请证书"按钮

                            
4. 等待证书申请完成

                        

                    

                

            

        


        
        

            

                
安装说明

                

                    
系统要求

                    
    
                        
  • CentOS 7+ / Ubuntu 18.04+
    • 
                        
  • Python 3.6+
    • 
                        
  • Nginx
    • 
                    


                    
安装步骤

                    
# 1. 下载项目
git clone https://github.com/your-repo/nginx-manager.git
cd nginx-manager

# 2. 运行安装脚本
chmod +x scripts/install.sh
./scripts/install.sh

# 3. 启动服务
source venv/bin/activate
./scripts/start.sh
                    


                    

                        注意：安装过程需要root权限
                    

                

            

        


        
        

            

                
卸载说明

                

                    

                        警告：卸载将删除所有配置和数据，请谨慎操作！
                    


                    
卸载步骤

                    
# 1. 停止服务
systemctl stop nginx

# 2. 运行卸载脚本
./scripts/uninstall.sh

# 3. 删除目文件
rm -rf /path/to/nginx-manager
                    


                    

                        提示：卸载前请确保已备份重要数据
